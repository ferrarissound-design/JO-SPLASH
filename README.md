# JO-SPLASH — CHROMA DUEL

Three.js 製、ブラウザで動作する 1 対 1 のインク陣取り FPS ゲームです。
床をインクで塗り合い、制限時間(90秒)終了時により広い面積を塗っていた側が勝利します。

## 起動方法

ビルドツールを使わない素の ES Modules 構成です。ブラウザの `fetch`/モジュール読み込みは
`file://` では動作しないため、任意の静的ファイルサーバーで配信してください。

```bash
# 例1: Python
python3 -m http.server 8080

# 例2: Node (npx)
npx serve .
```

起動後、ブラウザで `http://localhost:8080/index.html` を開いてください。

## 操作方法

| 操作 | 内容 |
| --- | --- |
| W / A / S / D | 移動 |
| マウス | カメラ操作・照準 |
| 左クリック | インク発射 |
| E | インクボム |
| 1 / 2 / 3 | メイン武器切替（連射 / 拡散 / 精密） |
| Space | ジャンプ |
| W + Space（自分色に塗った壁際で） | ウォールクライム（塗った壁をよじ登る） |
| Shift | 自分色のインクに潜伏（高速移動・隠密・急速回復） |
| 方向入力 + Shift + Space（潜伏中） | インクロール（高速回避・短時間のダメージ軽減） |
| Q | スペシャルゲージ満タン時にインクバースト |
| Esc | マウスロック解除 |
| R | ゲーム終了後のリスタート |
| `` ` `` (Backquote) | デバッグオーバーレイ表示切替 |
| V | （デバッグ）敵の外見タイプを切替（Speed → Street → Heavy → Technical） |

```
  entities/

    Player.js             プレイヤー入力処理

    PaintSystem.js        塗装グリッド + CanvasTexture 描画

    Weapon.js             発射制御（クールダウン・インク消費）
    ProjectileManager.js  インク弾のオブジェクトプール・衝突判定

```

## ステージの仕組み（塗って移動するマップ）

- 床に加えて、中央高台の上面とランプの斜面も同じ塗装グリッド／テクスチャを共有しており、通常の床と全く同じように撃って塗れる。プレイヤーの移動トレイル・インクサーフ・撃破時の環境ダメージもこれらの面で正しく機能する。
- 中央高台の全側面、箱型・円柱型の全障害物、アリーナ外周の内壁を塗装できる。各面は `WallPanel.js` の独立した塗装グリッドを持ち、通常弾は着弾点を、PRECISIONのチャージ弾は縦に連続した塗り筋を残す。
- 高台と障害物は、足元から頂上まで自分色のインクが連続していれば、壁に向かって **W + Space** でウォールクライムできる。箱・円柱の天面にも着地して移動できる。外周壁はアリーナ外へ出ないよう、塗装のみ可能でクライム対象外。
- 自分色の床で **Shift** を押すとインクへ潜伏する。潜伏中は射撃できない代わりに、高速移動、インクとHPの急速回復、低い当たり判定を得る。静止すると外見と敵マーカーがほぼ消え、移動中は飛沫で居場所が分かる。
- カメラは右肩後方から狙う三人称視点。中央照準へ向けてキャラクターの右肩から弾を発射し、壁際ではカメラが自動的に手前へ寄って地形へのめり込みを防ぐ。
- 主人公は大きな頭、目元マスク、ヒレ状の耳、頭頂部から前髪・後ろ髪まで連続した揺れる触手ヘア、ストリートウェア、スニーカー、背中のインクタンクを備えたオリジナルのインクスポーツ風デザイン。移動時は髪・武器・タンクが連動して揺れる。
- 主人公の脚・靴・腕・手は独立した関節リグで動き、低速時は小さな歩幅、高速時は膝上げ・脚の開き・靴の蹴り返し・腰の上下動を強めた走りへ連続的に変化する。ジャンプ・潜伏・ウォールクライムにも専用ポーズがある。
- 潜伏中に方向入力＋**Space**でインクロールを発動する。短い高速跳躍中は被ダメージを60%軽減し、着地後はすぐ反撃へ移れる。連発防止の0.85秒クールダウン付きで、デバッグ表示中は **G** で強制確認できる。
- 撃破時は倒した側の大きなインク爆発が足元に残るため、対人戦の勝敗がそのまま陣取りにも影響する。
- 新しく自分の色へ塗り替えたセル数に応じてスペシャルゲージが溜まる。満タン時に **Q**（タッチ操作は **BURST**）でインクを全回復し、周囲へ広がる大きな塗装波と対人ダメージを発生させる。
- メイン武器は連射型の **STREAM**、近距離で横に広く塗る **SPREAD**、チャージ型の **PRECISION** の3種。PCでは **1 / 2 / 3** で切り替えられる。PRECISIONは射撃を長押ししてチャージし、離すと蓄積率に応じて威力・弾速・射程・塗り幅が伸びる。チャージ弾は射線下の塗れる床へ連続したインクラインを残し、塗れる壁へ当たると壁面に長い塗り筋を作る。満充電後に自分のインクへ潜ると1.6秒だけチャージを保持し、浮上後に射撃を押し直して離すことで位置を変えてから撃てる。サブ武器のインクボムは **E**（タッチ操作は **BOMB**）で使用する。
- 中央高台は旧バージョンより一回り大きくし（`ARENA.platformSize`）、面積そのものの制圧価値を上げた。周囲には遮蔽物を追加し、撃ち合いだけでなく回り込みを誘発する。
- 各スポーン地点の周囲には試合開始時とリスポーン直後に自陣色で塗られた安全地帯（`ARENA.spawnSafeRadius`）が用意され、復帰直後にいきなり中立／敵地に立たされることはない。
- 中央への4方向の進入路には、ジャンプで越えられる腰高の遮蔽物を配置。射線を切って塗り直すか、飛び越えて短距離を詰めるかを選べる。
- CPUは目的地への直進だけでなく、進行方向を複数サンプリングし、自陣インクを優先・敵インクを回避する。特に撤退とインク補給時は安全な塗装ルートを強く選ぶ。
- CPUも高台・障害物の壁ルートを選択し、壁へ接近して下から上まで塗装したあと、プレイヤーと同じ縦の塗り筋判定・インク消費でクライムする。デバッグ表示中は **L** でこの行動を強制できる。
- 中央高台はランプ開口部の左右を含む全側面が塗れるため、ランプ以外の方向からも塗装ルートを作って進入できる。
- CPUは交戦距離に応じてSPREAD・STREAM・PRECISIONを切り替える。遮蔽物越しや高台のプレイヤーには弾道を計算したインクボムを優先し、通常戦闘でも低確率で制圧目的に投げる。デバッグ表示中は **B** でボム、**K** で武器切替を強制できる。
- CPUも塗装とKO塗り返しでスペシャルを蓄積し、塗装劣勢・低HP・近距離での圧力を条件にインクバーストを使用する。発動前には0.75秒の警告が表示され、デバッグ表示中は **O** で発動確認できる。
- タイトル画面でCPU難易度を **ROOKIE / STANDARD / ELITE** から選択できる。判断間隔・照準誤差・移動速度・ボム頻度・スペシャル回転が連動して変化し、再戦時も選択を保持する。
- HUD左上のライブ塗装マップで、床の塗り分けと双方の向き・位置をリアルタイムに確認できる。CPUがインクへ潜伏して静止している間はマーカーが消え、索敵の駆け引きを維持する。
- 残り10秒から専用の秒読み・加速BGM・警告音で終盤を強調し、0秒では **TIME UP** 後に1.4秒の集計演出を挟んで結果を発表する。デバッグ表示中は **T** で残り12秒へ短縮して確認できる。

## 現在の制限事項

- ステージは 1 種類のみ（小型アリーナ）
- 壁の塗装は各面ごとの独立グリッドで管理され、床の塗装率には加算されない
- セーブ／永続化機能はなし（1 セッション完結）
